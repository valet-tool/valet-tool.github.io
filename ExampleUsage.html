Example Usage

This page will describe basic usage and implementation of the data into self-created and existing simulation tools. While we will describe some basic applications of our VALET data, there are many other existing simulation tools and resources that our data may be integrated into. One example of this is the Exemplars resource.

Implementation Into Self-Created Tool

We created a basic system that includes examples of A) Not accounting for tactic volatility B) Using a small set of volatile tactic data to demonstrate the impact that tactic volatility may have on the determined utility. Implementations of utility equations are likely to be domain specific. Therefore, our intention for this example was to merely create a very basic, example utility equation to demonstrate the future benefits of our work and accounting for tactic volatility. Future work may include the inclusion of a basic machine learning algorithm to predict tactic values in this example.
